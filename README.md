## 🚀 Quick Start

```bash
# 1. Clone
git clone https://github.com/Itachi-0xAI/gitroast.git
cd gitroast
npm install

# 2. Add your free Groq API key (console.groq.com — no credit card)
cp .env.example .env.local
# → paste GROQ_API_KEY=gsk_... in .env.local

# 3. Run
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and roast yourself first.

## 🔑 Environment Variables

| Variable | Required | Where to get it |
|----------|----------|-----------------|
| `GROQ_API_KEY` | ✅ Yes | [console.groq.com](https://console.groq.com) — free |
| `GITHUB_TOKEN` | ❌ No | Boosts rate limit from 60 → 5000 req/hr |
